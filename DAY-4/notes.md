#  Python Lists Notes

##  1. What is a List?
- A list is a collection of items stored in a single variable.
- Lists are ordered and changeable (mutable).
- They can store multiple data types (numbers, strings, etc.).
- Lists allow duplicate values.
- Defined using square brackets.

---

##  2. Accessing List Elements
- Elements in a list are accessed using their index.
- Index starts from 0 (first element).
- Negative indexing is also supported (starts from the end).
- Helps retrieve specific values from the list.

---

##  3. Modifying Lists
- Lists are mutable, so their elements can be changed.
- You can update, add, or remove elements.
- Changes directly affect the original list.
- Useful for dynamic data handling.

---

##  4. Slicing Lists
- Slicing is used to access a portion of a list.
- It allows selecting a range of elements.
- Uses start, stop, and step values.
- Does not modify the original list, returns a new list.

---

##  5. List Functions and Methods
- Lists have built-in functions and methods to perform operations.
- Common uses:
  - Adding elements
  - Removing elements
  - Sorting elements
  - Finding length
  - Counting occurrences
- These methods make list handling easy and efficient.

---

##  6. Nested Lists
- A nested list is a list inside another list.
- Used to represent complex data structures (like matrices).
- Elements inside nested lists can be accessed using multiple indices.
- Helpful for multi-dimensional data.

---

##  Summary
- Lists store multiple values in one variable.
- They are ordered, mutable, and allow duplicates.
- Elements can be accessed, modified, and sliced.
- Built-in methods simplify operations.
- Nested lists help manage complex data.
