#  Python Tuples & Sets Notes

##  1. Tuples in Python
- A tuple is a collection of items stored in a single variable.
- Tuples are ordered and unchangeable (immutable).
- They allow duplicate values.
- Defined using parentheses.
- Faster than lists and used for fixed data.

---

##  2. Accessing Tuple Elements
- Elements are accessed using indexing.
- Index starts from 0.
- Negative indexing is supported (from the end).
- Used to retrieve specific values.

---

## 3. Tuple Operations
- Tuples support operations like:
  - Concatenation (joining tuples)
  - Repetition
  - Membership checking
- Since tuples are immutable, elements cannot be modified.

---

##  4. Tuple Methods
- Tuples have limited built-in methods.
- Common uses:
  - Counting occurrences of elements
  - Finding the index of an element
- Mainly used for accessing and analyzing data.

---

# 🔹 Sets in Python

##  5. Sets in Python
- A set is a collection of unique elements.
- Sets are unordered and mutable.
- Duplicate values are automatically removed.
- Defined using curly braces.
- Useful for storing distinct data.

---

## 6. Set Operations
- Sets support mathematical operations:
  - Union → Combines elements from both sets
  - Intersection → Common elements
  - Difference → Elements in one set but not the other
  - Symmetric Difference → Elements in either set but not both
- Useful in data comparison and filtering.

---

##  7. Set Methods
- Sets provide built-in methods for:
  - Adding elements
  - Removing elements
  - Updating sets
  - Checking membership
- Efficient for operations involving unique data.

---

##  Summary
- Tuples are ordered and immutable collections.
- They are used for fixed data and faster performance.
- Sets store unique elements and are unordered.
- Set operations help in comparing and combining data.
- Both tuples and sets are important data structures in Python.
