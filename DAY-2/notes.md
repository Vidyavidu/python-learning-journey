# 🐍 Python Day-2 Notes

## 📌 1. Input and Output in Python
- Input is used to take data from the user during program execution.
- Output is used to display results to the user.
- Input data is usually taken as text and may need type conversion.
- Output can display text, variables, or formatted data.
- Proper input and output handling improves user interaction.

---

##  2. String Manipulation
- A string is a sequence of characters enclosed in quotes.
- Strings are immutable (cannot be changed after creation).
- Common operations:
  - Concatenation (joining strings)
  - Repetition
  - Indexing (accessing characters)
  - Slicing (getting a part of the string)
- Strings support various built-in methods for formatting and processing.
- Useful for handling text data in programs.

---

##  3. Comments in Python
- Comments are used to explain code and improve readability.
- They are ignored during program execution.
- Types of comments:
  - Single-line comments
  - Multi-line comments
- Useful for documentation and understanding logic.
- Helps in debugging and maintaining code.

---

##  4. Escape Sequences
- Escape sequences are special characters used inside strings.
- They start with a backslash (\\).
- Used to represent characters that are difficult to type directly.

### Common Escape Sequences:
- \\n → New line
- \\t → Tab space
- \\' → Single quote
- \\" → Double quote
- \\\\ → Backslash

- Helpful in formatting output and handling special characters in strings.

---

##  Summary
- Input and output handle user interaction.
- Strings are important for text processing.
- Comments improve code readability and maintenance.
- Escape sequences help format and manage special characters in strings.
