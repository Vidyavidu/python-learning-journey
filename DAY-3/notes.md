#  Python Operators Notes

##  1. Arithmetic Operators
Used to perform mathematical operations on values.

- Addition → Adds two values  
- Subtraction → Subtracts one value from another  
- Multiplication → Multiplies values  
- Division → Divides values and returns float result  
- Modulus → Returns remainder  
- Exponentiation → Raises to power  
- Floor Division → Returns integer result (removes decimal part)

---

##  2. Assignment Operators
Used to assign values to variables.

- Basic assignment assigns a value to a variable  
- Combined assignment operators perform operation and assignment together  
- Helps reduce code length and improve readability  
- Commonly used in updating variable values

---

##  3. Comparison Operators
Used to compare two values.

- Returns either True or False  
- Helps in decision-making (conditions)  
- Checks equality, inequality, greater than, less than, etc.  
- Widely used in conditional statements like if-else

---

##  4. Logical Operators
Used to combine multiple conditions.

- Works with Boolean values (True/False)  
- Used in decision-making and control flow  
- Types:
  - AND → True if both conditions are true  
  - OR → True if at least one condition is true  
  - NOT → Reverses the result  

---

##  5. Membership Operators
Used to check whether a value exists in a sequence.

- Works with strings, lists, tuples, etc.  
- Returns True if value is present, otherwise False  
- Helps in searching and validation tasks  

---

##  6. Bitwise Operators
Used to perform operations on binary (bit-level) values.

- Works on integers  
- Operates bit by bit  
- Used in low-level programming and optimization  
- Includes operations like AND, OR, XOR, NOT, shift operations  

---

##  Summary
- Arithmetic operators perform calculations  
- Assignment operators store and update values  
- Comparison operators compare values and return Boolean results  
- Logical operators combine conditions  
- Membership operators check presence in sequences  
- Bitwise operators work at binary level
