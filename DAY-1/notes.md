# 🐍 Python Day-1 Notes

##  1. Variables in Python
- Variables are used to store data values.
- Python does not require explicit declaration of variable types.
- The type of a variable is decided automatically at runtime.
- Variable names should be meaningful and follow naming rules (no spaces, cannot start with numbers, etc.).

---

##  2. Data Types in Python
Python has several built-in data types:

- **int** → Integer numbers (e.g., 10, 100)
- **float** → Decimal numbers (e.g., 10.5, 3.14)
- **str** → Text/String (e.g., "Hello")
- **bool** → Boolean values (True or False)
- **list** → Ordered, changeable collection
- **tuple** → Ordered, unchangeable collection
- **dict** → Key-value pairs

---

##  3. Type Conversion
- Type conversion means changing one data type into another.
- Two types:
  - **Implicit Conversion** → Done automatically by Python
  - **Explicit Conversion** → Done manually using functions
- Helps in performing operations between different data types.

---

##  4. Arithmetic Operators
Used to perform mathematical operations:

- **Addition (+)**
- **Subtraction (-)**
- **Multiplication (*)**
- **Division (/)**
- **Modulus (%)** → Gives remainder
- **Exponentiation (**)** → Power
- **Floor Division (//)** → Removes decimal part

---

##  5. Assigning Values to Multiple Variables
- Python allows assigning the same value to multiple variables at once.
- It also allows assigning different values to multiple variables in a single line.
- This makes code shorter and more readable.

---

##  6. Variable Reassignment
- Variables can be updated or changed at any time.
- The new value replaces the old value.
- Python allows changing both value and type during reassignment.

---

##  Summary
- Variables store values dynamically.
- Python supports multiple built-in data types.
- Type conversion helps manage data compatibility.
- Arithmetic operators perform calculations.
- Multiple assignments make coding efficient.
- Variables can be reassigned anytime.
